# 🏥 Hospital Appointment Scheduling & Telemedicine Management System  

## 📌 Project Overview  
The **Hospital Appointment Scheduling and Telemedicine Management System** is a **Java Full Stack web application** designed to streamline healthcare services by integrating **appointment booking, patient management, doctor management, and telemedicine (video consultation)** into a single platform.  

This system reduces manual effort, improves efficiency, and provides **remote healthcare accessibility** for patients.

---

## 🚀 Features  

- 🔐 Secure User Authentication (Patient & Doctor Login/Register)  
- 👨‍⚕️ Doctor Management with specialization & availability  
- 🧑‍🤝‍🧑 Patient Profile & Medical History Management  
- 📅 Smart Appointment Scheduling System  
- 📹 Telemedicine (Video Consultation)  
- 📊 Patient & Doctor Dashboards  
- 📝 Consultation Reports & Reviews  
- 🗄️ Centralized Database Management (MySQL)
